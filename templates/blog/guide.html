{% if lang == 'id' %}Daftar Akun Baru{% else %}Register New Account{% endif %}

{% if lang == 'id' %} Buka halaman registrasi dan isi form dengan data Anda. Gunakan email aktif untuk verifikasi. After login, you will be redirected to the main dashboard. {% endif %}
